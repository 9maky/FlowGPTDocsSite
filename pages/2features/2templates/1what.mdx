## What are templates on FlowGPT?

A template is a pre-made prompt that can be used to generate text, translate languages, write different kinds of creative content, and answer your questions in an informative way.

Templates can be a great way to save time and effort when using FlowGPT. For example, if you need to write a blog post about the benefits of using FlowGPT, you can use a template to get started. Or, if you need to translate a document from English to Spanish, you can use a template to do that.

Templates are also a great way to learn how to use FlowGPT. By using templates, you can see how other people have used FlowGPT to generate text, translate languages, write different kinds of creative content, and answer questions in an informative way.

To use a template on FlowGPT, you will need to follow these steps:

1. Go to the [FlowGPT website](https://flowgpt.com/).
2. Click on the "Browse a Prompt" button.
3. In the "Search" field, type in the name of the template prompt that you are looking for. Keep in mind that not all prompts are template prompt.
4. Click on the "Search" button.
5. The list of prompts will be displayed.
6. Click on the name of the template prompt that you want to use.
7. The template mode will be displayed.
