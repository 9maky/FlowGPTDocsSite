## How to upvote prompts

**Overview**

You can upvote prompts that you like so that they can be seen by more people. To upvote a prompt, follow these steps:

1. Find the prompt that you want to upvote.
2. Click the **Upvote** button next to the save button.

**Steps**

1. Find the prompt that you want to upvote. You can do this by browsing the prompts page, searching for a specific prompt, or following a link to a prompt.
2. Click the **Upvote** button next to the save button. The **Upvote** button is a gray button with a white arrow pointing up.

**Tips**

- You can upvote multiple prompts.
- You can upvote prompts that you have saved.
- You can upvote prompts that you have created.

**Troubleshooting**

If you are having trouble upvoting prompts, here are some things to check:

- Make sure that you are logged in to your account.
- Make sure that you have clicked the **Upvote** button.

If you are still having trouble upvoting prompts, please contact support.
